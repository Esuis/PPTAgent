# PPT生成平台 — 用户操作手册

## 一、平台简介

PPT生成平台是一个智能PPT制作系统，您只需通过浏览器输入主题和要求，系统即可自动完成资料搜索、内容撰写和版式设计，生成专业美观的PPT文件。

**核心能力：**
- 自动搜索并整理资料
- 智能撰写幻灯片内容
- 自动设计版式与配色
- 支持上传参考文档作为素材来源
- 实时预览生成过程
- 一键下载 .pptx 文件

---

## 二、访问方式

在浏览器中打开系统地址即可使用（推荐使用 Chrome、Edge 等现代浏览器）。

> 具体访问地址请向管理员获取。

## 四、操作步骤

### 第1步：选择页数

在输入区域上方，点击 **页数选择框**，设定您希望生成的幻灯片数量：

| 选项 | 说明 |
|------|------|
| 自动页数（Auto） | 由系统根据内容自动决定页数（推荐） |
| 指定页数（1-30页） | 手动指定生成多少页幻灯片 |

### 第2步：输入主题

在输入框中，输入您的PPT主题和要求。请尽量描述清楚，系统将根据您的描述生成内容。

**输入示例：**

```
介绍中国高铁发展历程，生成10页PPT
```

```
基于现有知识介绍交通银行的发展历程和业务范围，生成8页ppt，禁止联网
```

```
为公司年会制作一份总结PPT，包含年度业绩回顾、团队成果展示、明年规划展望三个部分
```

**输入技巧：**
- 明确主题：清楚说明PPT要讲什么
- 指定页数：如需特定页数，请在文本中说明
- 添加限制：如"禁止联网"则系统不会搜索外部资料
- 提供结构：可以列出希望的章节或要点

> 输入框最多支持 2000 个字符。

### 第3步：上传附件（可选）

如需基于已有文档生成PPT，可点击 **上传附件** 按钮上传参考文件。

**支持的文件格式：**

| 格式 | 说明 |
|------|------|
| .txt | 纯文本文件 |
| .md | Markdown文件 |
| .docx | Word文档 |
| .pdf | PDF文件 |

**注意事项：**
- 每次最多上传 **1个** 附件
- 附件大小不超过 **3MB**
- 上传附件后，系统会从文档中提取内容作为PPT的素材来源

### 第4步：点击发送

确认主题和设置后，点击 **发送** 按钮（或按 Enter 键）提交任务。

## 五、生成过程

### 5.1 进度展示

发送后，左侧聊天区域会实时显示生成进度，包括：

- 启动任务，正在准备...
- 搜索资料（系统搜索相关信息时）
- 读取文件 / 写入文件
- 生成第 N 页幻灯片
- 编辑第 N 页幻灯片

### 5.2 实时预览

右侧预览区域会在幻灯片生成过程中 **逐页显示**：

- 顶部显示已生成的页数（如"已生成 5 页"）
- 缩略图导航条可快速跳转到任意页
- 中间主区域展示当前选中页的完整效果
- 底部可通过 **上一页 / 下一页** 按钮翻页浏览

### 5.3 排队等待

当系统繁忙时，您可能需要排队等待。排队时会出现弹窗提示：

- 显示当前位置（如"第 3 位"）
- 位置会实时更新
- 可点击 **取消排队** 退出等待
- 轮到您时自动开始生成，无需任何操作

---

## 六、下载文件

生成完成后：

1. 聊天区域会提示"幻灯片生成完成"
2. 右侧预览区顶部出现 **下载按钮**（↓图标）
3. 点击下载按钮，即可获取 .pptx 格式的PPT文件

> 下载按钮仅在生成完成后才可点击。

## 七、常见问题

### Q1：生成需要多长时间？
生成时间取决于PPT页数和内容复杂度，通常需要几分钟。页数较多或内容需要联网搜索时，耗时会更长。

### Q2：生成过程中可以关闭页面吗？
不建议关闭。生成过程中系统通过实时连接推送进度，关闭页面会导致连接中断，任务将被自动取消。

### Q3：上传的文档内容会完整出现在PPT中吗？
系统会对文档内容进行智能提取和整理，根据PPT的呈现方式进行改写和精简，不会原文照搬。

### Q4：提示"您已有任务正在执行中"怎么办？
同一账号同时只能执行一个生成任务，请等待当前任务完成后再提交新的任务。

### Q5：生成结果不满意怎么办？
您可以重新提交一个新的生成任务，在输入中更详细地描述您的需求，包括：
- 更具体的主题要求
- 期望的章节结构
- 特定的内容要点
- 期望的页数

### Q6：可以用手机访问吗？
系统支持移动端浏览器访问，界面会自动适配屏幕大小。但建议使用电脑端以获得最佳体验。

### Q7：附件上传失败怎么办？
请检查：
- 文件格式是否为 .txt / .md / .docx / .pdf
- 文件大小是否超过 3MB
- 是否已有其他附件（每次仅限1个）

### Q8：生成中断了怎么办？
如果生成过程中出现错误，聊天区域会显示错误信息。请重新提交任务即可。

## 八、最佳实践

1. **主题明确**：输入越清晰，生成效果越好。避免过于笼统的描述。
2. **善用附件**：有现成资料时，上传文档比纯文字描述效果更好。
3. **合理指定页数**：内容多时可适当增加页数，避免内容被过度压缩。
4. **分次生成**：如需制作多个不同主题的PPT，请逐个完成，不要同时提交。
5. **先预览后下载**：生成完成后先在右侧逐页预览效果，确认满意后再下载。
